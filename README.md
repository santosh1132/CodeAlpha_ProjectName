

# Titanic Data Analysis and Visualization

## Project Overview
This project analyzes the Titanic dataset and presents meaningful insights using Python. The dataset was explored, cleaned, and visualized to understand passenger information, survival patterns, and data distribution.

## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data exploration
- Data structure inspection
- Missing value analysis
- Statistical summary
- Pattern identification
- Data visualization

## Visualizations
- Survival Count Bar Chart
- Survival Percentage Pie Chart
- Passenger Distribution Histogram
- Outlier Detection Box Plot

## Dataset
Titanic Dataset
